# [tm-scoreboard](https://vodbox.github.io/tm-scoreboard/)
Taskmaster Scoreboard Generator with Saved State

Forked from [tm-scoreboard](https://github.com/vodbox/tm-scoreboard) to add:

1) Local browser storage for offline and persistent saved scoreboards (including images)
2) Multiple games stored with options to resume, delete, or start new games
3) Allow user to change background
4) Added `pnpm run dev` for development
